# Mirkvartir Scraper & DVC Pipeline

## Описание проекта
Скрейпер для сбора данных о недвижимости с сайта [mirkvartir.ru](https://www.mirkvartir.ru/). 
Проект реализует автоматизированный сбор трех типов данных:
* **Структурированные признаки:** цена, площадь, этажность, координаты.
* **Текст:** описания объектов.
* **Медиа:** автоматическая загрузка изображений объектов.

## Технологический стек
* **Управление проектом:** `uv` (зависимости, виртуальное окружение).
* **Сбор данных:** `requests`, `BeautifulSoup`, `re`.
* **Версионирование данных:** `DVC` (Data Version Control) для хранения изображений вне Git.

## Инструкция по запуску
1. Установка окружения: `uv sync`
2. Запуск скрейпера: `uv run src/my_project/data/scraper.py`
3. Получение данных из DVC: `dvc pull`
